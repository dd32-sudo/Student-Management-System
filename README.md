# Student-Management-System


A Python console application for managing student grades, built as a self-learning project.

## Features / 功能
- Input validation for name, student ID, and marks
- Continuous Assessment calculation (Test 40% + Project 30% + Workshop 30%)
- Module mark calculation (CA 50% + Exam 50%)
- Auto grading (A/B/C/F) with personalised comments
- F-grade safeguard: auto-fail if CA < 40 or Exam < 40
- Batch statistics: average marks, grade distribution
- Quick Grade Check mode: input a single score to see the grade instantly

## Built With / 使用技術
- Python
